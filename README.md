## Run Locally

1. **Clone the repo**
```bash
git clone https://github.com/sohaibmohd18/sohaibmohammed.portfolio.git
cd sohaibmohammed.portfolio.git
```

2. **Install dependencies**
```bash
npm install
```

3. **Run the development server**
```bash
npm run dev
```
